Put.io XBMC plugin
by Maximilian Szengel (maxsz)

Browse and stream from put.io with your XBMC.

Dependencies

  • XBMC
  • script.module.simplejson (Should install automatically with plugin)

Install

Three easy steps:
  • Download as ZIP
  • Start XBMC and navigate to System->Addons->Install from zip. Select the downloaded ZIP and install.
  • Enter your credentials. Make sure you enter the api key/secret and not your login information. You can find your api key/secret on your put.io settings page.
